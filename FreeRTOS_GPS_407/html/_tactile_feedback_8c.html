◆ BackWaypoint()

int BackWaypoint ( void )

BackWaypoint wordt gebruikt om terug te gaan naar de vorige waypoint. Deze functie wordt aangeroepen door de 0ARM-keys. In de functie is een check ingebouwd die kijkt of de functie correct is uitgevoerd.

  • Returns
    1 = waypoint verlaagt, verlaag CurrentWaypoint met 1 voor vorige waypoint. 0 = waypoint niet verlaagt.
