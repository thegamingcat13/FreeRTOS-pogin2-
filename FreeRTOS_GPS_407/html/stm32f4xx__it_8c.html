void EXTI0_IRQHandler ( void )

This function handles EXTI line0 interrupt.

Deze interrupthandler wordt automatisch gegenereerd nadat de interrupt aangezet is op de processor via .ioc. De inhoud van de handler moet je natuurlijk zelf nog coderen. Hier vangt de handler de toetsen van het ARM-keyboard op.
Nadat een toets gedrukt is, wordt bepaald WELKE toets dat was met KEYS_read(). Vervolgens wordt de key meegestuurd met een FreeRTOS event, die opgevangen wordt door de task ARM_keys_IRQ().
