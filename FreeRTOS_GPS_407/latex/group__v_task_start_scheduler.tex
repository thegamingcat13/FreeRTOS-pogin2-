\begin{DoxyPre}void vTaskStartScheduler( void );\end{DoxyPre}


Starts the real time kernel tick processing. After calling the kernel has control over which tasks are executed and when.

See the demo application file \doxylink{main_8c}{main.\+c} for an example of creating tasks and starting the kernel.

Example usage\+: 
\begin{DoxyPre}
void vAFunction( void )
\{
    // Create at least one task before starting the kernel.
    xTaskCreate( vTaskCode, "{}NAME"{}, STACK\_SIZE, NULL, tskIDLE\_PRIORITY, NULL );

    // Start the real time kernel with preemption.
    vTaskStartScheduler ();

    // Will not get here unless a task calls vTaskEndScheduler ()
\}
  \end{DoxyPre}
