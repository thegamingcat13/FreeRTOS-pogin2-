void EXTI0\+\_\+\+IRQHandler (\begin{DoxyParamCaption}\item[{void}]{}{}\end{DoxyParamCaption})}



This function handles EXTI line0 interrupt. 

Deze interrupthandler wordt automatisch gegenereerd nadat de interrupt aangezet is op de processor via .ioc. De inhoud van de handler moet je natuurlijk zelf nog coderen. Hier vangt de handler de toetsen van het ARM-\/keyboard op. ~\newline
 Nadat een toets gedrukt is, wordt bepaald WELKE toets dat was met KEYS\+\_\+read(). Vervolgens wordt de key meegestuurd met een Free\+RTOS event, die opgevangen wordt door de task \doxylink{admin_8h_a0c660e87fd3fb5216355c32e3a72f95e}{ARM\+\_\+keys\+\_\+\+IRQ()}.
